¿Qué es Data Manager?

Data Manager es un proyecto que busca proveer una gran variedad de servicios construidos por los mismos usuarios de la plataforma. Queremos construir una comunidad dispuesta a aportar ideas y conocimientos para así crear y mejorar servicios. Si bien el proyecto está orientado a programadores dado que estos son los creadores de servicios, en un futuro vamos a incorporar interfaces para usuarios que no sepan programar.

¿Qué es un servicio?

Un servicio es una página cuya funcionalidad es administrar cierta información. Dicha información puede ser de cualquier categoría. De esta manera, cada servicio se convierte en una SPA (Single Page Application) ya que se basa en un solo archivo html (obviamente, el cual puede importar otros archivos tales como css, js, etc.).

Creá y personalizá tus propios servicios!

¿Querés crear un servicio? Seguí estos pasos:

  • Entrá a nuestro repo.
  • Hacé un fork.
  • Agregá tu servicio mediante un archivo html en ./src/pages/ (y los archivos que necesites en ./src/styles/ y ./src/js donde corresponda).
  • Hacé un pull request. Ni bien comprobemos el pull request, lo aceptaremos.

¿Querés saber cuando surgen nuevos servicios?

Ingresá tu email para recibir las últimas novedades!
